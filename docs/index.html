Termos de Uso
000-1 Acesso aos arquivos local

O acesso aos arquivos locais são solicitados ao iniciar o software e são criadas pastas no armazenamento interno do dispositivo para logs do sistema e informações para prestação de suporte

000-2 Negação de serviço

Todo acesso requisitado negado irá gerar uma negação de serviço, bem como o mal funcionamento da aplicação podendo ocorrer bugs, então é recomdavél a permissão do acesso assim que solicitado.

000-3 Politica de dados

Os dados fornecidos ao sistema irão e deverão estar em controle do utilizador por todo o tempo, passando transaparência na utilização, todos os dados podem ser excluídos permanentemente e mofificados, nenhum dado é realocado em servidores externos, resalvo caso de backup, ao contrario serão mantidos no dispositivo até que seja solicitado o pedido de backup.

000-4 Localização e captura de dados

Acesso a localização é requesitado e pode ser negado,se for negado,o dado deverá ser passado manualmente, os dados capturados para o funcionamento da aplicação, são enviados a google para que seja possível a integração com o mapa, seguindo a politica da empresa em questão.

000-5 Dados de terceiros

A intrudução de informação de terceiros será deletada depoua do uso da informação, afim de manter a privacidade do cliente.
